16-bit Adder/Subtractor

The below circuit diagram shows a 4-bit adder/subtractor. I implemented a 16-bit adder/subtractor using strucutral Verilog to carry out addition and subtraction with hexadecimal numbers (code available upon request).

The 16-bit adder/subtractor was implemented with Verilog HDL using a series of 16 1-bit full adder modules that are connected sequentially to process each bit of the inputs. The operation done was decided on by an opcode value. If the opcode was 0, then addition was done by simply adding the input values. If the opcode was 1, then the second value had its bits inverted and the carry-in was set to 1 to perform two's complement. Each 1-bit full adder outputs a sum and a carry-out, which is fed into the next adder. Functional and unit gate delay simulations were done to verify accuracy and timing behavior.
