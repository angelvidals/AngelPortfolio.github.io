Interactive Data Visualization for Incident Analysis

Using a CSV file with real-world data about crimes in the city of Buffalo, I extracted key information. I then displayed that information with graphs and charts on a webpage using Plotly, JavaScript, HTML, and Python (code available upon request).

The user interface is built using HTML and JavaScript, with integration of the Plotly library for data visualization. The frontend features an input field for users to specify an hour of the day and a button that triggers an AJAX call to the server. Data is collected from the CSV file and a line graph shows incidents that occured at the given hour over the years.

The backend is implemented in Python, where requests are processed. Data is parsed and read from a CSV file, which serves as the source of incident information. The processed data is sent back to the frontend via AJAX requests, where it is rendered into charts and graphs using Plotly. The backend is set up with RESTful endpoints that respond to GET and POST requests, allowing the frontend to fetch processed data.
